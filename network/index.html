# Computer Network

# 301、302、303、307 是什么？

  • 301 Moved Permanently（永久重定向）：
    • 表示请求的资源已经被永久移动到新的URL。
    • 客户端应该使用新的URL进行以后的请求。
    • 搜索引擎会将旧URL的权重转移到新URL上。
  • 302 Found（临时重定向）：
    • 表示请求的资源暂时被移动到另一个URL。
    • 客户端应该继续使用原来的URL进行以后的请求。
    • 这是HTTP/1.0的标准，在HTTP/1.1中建议使用303或307来明确指示客户端应该使用GET方法还是继续使用原来的方法。
  • 303 See Other （查看其他位置）å
    • 表示请求的资源可以在另一个 URL 找到，应该使用GET方法进行请求
    • 主要用户响应 POST、PUT 等请求，告诉客户端使用GET方法去获取新的资源
  • 307 Temporary Redirect（临时重定向）：
    • 表示请求的资源暂时被移动到另一个URL，但客户端应继续使用原来的HTTP方法（如POST）进行请求。
    • 与302不同，307明确保持了原始请求方法。
